#RTCHose - Twitter bot

This is my experiment on implementing a (re)Twitter bot.

It is a node.js application.

Before run:

`npm install`

Also create a `config.json` file which contain:

```json
{ 
  "twitter": {
      "consumer_key":         "....."
    , "consumer_secret":      "....."
    , "access_token":         "..-.."
    , "access_token_secret":  "....."

  },
 "my_id": "twitter account id"
}
```

to run:
`node index.js`
